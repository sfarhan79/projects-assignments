Business Objective:

What is the probability of a business going bankrupt? Build a classification model to predict, considering the factors in the dataset.

● The variable to predict is binary (bankruptcy or non-bankruptcy).
● The dataset contains features for about 250 companies.

Perform:
1. Exploratory Data Analysis (EDA)
a. Describing the dataset
b. Cleaning the data
c. Checking invalid records
d. Missing value detection and imputation
e. Duplicated records
f. Outliers
g. Feature Engineering
h. Data Visualization

2. Model building
a. Identifying algorithms, building model/s, testing and evaluation.

3. Model Deployment
a. Utilization of pickle 
b. Leveraging Flask framework
c. Deploying via Stream-lit
